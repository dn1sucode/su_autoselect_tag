# Auto Tag Dimension Label

Автоматически назначает теги (слои) для размеров и текстовых меток в SketchUp.

## Что делает
- При появлении размеров (`Sketchup::Dimension`) присваивает тег `Dimension`.
- При появлении текстовых меток (`Sketchup::Text`) присваивает тег `Label`.
- Работает для геометрии в модели и в новых компонентных определениях.

## Установка
1. Сохраните файл `auto_tag_dimension_label.rb`.
2. Поместите его в папку плагинов SketchUp (например, `Plugins`).
3. Перезапустите SketchUp.

## Использование
Скрипт запускается автоматически при старте SketchUp и при открытии/создании модели.
Новых действий в интерфейсе не требуется.

## Настройка
Имена тегов можно изменить в начале файла `auto_tag_dimension_label.rb`:
- `TAG_DIMENSION = "Dimension"`
- `TAG_LABEL = "Label"`

## Совместимость
Используются наблюдатели SketchUp и проверка на разные API классы наблюдателей определений.
Скрипт рассчитан на работу в версиях SketchUp, где доступны соответствующие API классы.
